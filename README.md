# 2D Graphics Editor in C

## Description
A menu-driven 2D Graphics Editor implemented in C using a 2D character array as the drawing canvas.

## Features
- Draw Rectangle
- Draw Line
- Draw Triangle
- Draw Circle
- Display Canvas
- Clear Canvas
- Menu Driven Interface

## Technologies Used
- C Programming
- GitHub
- VS Code

## How to Run

gcc main.c graphics.c -o editor.exe

.\editor.exe
